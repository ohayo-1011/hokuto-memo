# 🌟 北斗メモ - スマスロ北斗の拳 実戦記録アプリ

スマホでスマスロ北斗の拳の実戦データを簡単に記録できるWebアプリ（PWA）です。
ホールでの使いやすさを重視したダークモードUIと、設定推測に役立つ機能を搭載しています。

## ✨ 主な機能

- **ワンタップ記録**: 謎当たり、チェリー、スイカなどを各種契機ボタンで素早く記録。
- **モード管理**: 「通常」「天国」「地獄」の状態をタップで切り替え、当選時の状態を保存。
- **小役カウンター**: 共通ベル、弱チェリー、スイカの確率をリアルタイム計算。
- **設定差情報**: 共通ベル確率やモード別当選率などの設定判別ポイントを表示。
- **履歴管理**: 過去の実戦データをブラウザに保存し、いつでも振り返り可能。
- **PWA対応**: ホーム画面に追加してアプリのように使えます（オフライン対応）。

## 🚀 使い方

### ローカルでの起動
Node.jsが必要です。

```bash
# 依存関係なし（静的サーバーのみ）
node server.js
```
http://localhost:3001 にアクセスしてください。

### Web公開（デプロイ）

VercelやNetlifyなどの静的ホスティングサービスへのデプロイがおすすめです。
GitHubリポジトリにプッシュし、連携するだけで公開できます。

## 📦 ファイル構成
- `index.html`: アプリ本体
- `css/style.css`: スタイルシート
- `js/app.js`: アプリロジック
- `js/storage.js`: データ保存ロジック
- `sw.js`: オフライン対応用 Service Worker
